# MIPEX-Visualization
Interactive dashboard visualizing the Migrant Integration Policy Index (MIPEX 2023) across EU countries.

The project combines an interactive map and dynamic radar charts to enable country-level analysis and cross-country comparison of eight integration policy dimensions: Labour Market Mobility, Family Reunification, Education, Political Participation, Permanent Residence, Citizenship, Antidiscrimination, Health.

Data Source: Migrant Integration Policy Index (MIPEX 2023) https://www.mipex.eu/
